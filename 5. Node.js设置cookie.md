1. 通过res对象设置响应头
2. 设置cookie时, 可以设置cookie**生效的路径path**
3. 为了防止用户修改cookie, 可以设置**httpOnly**
4. 设置cookie**过期时间**
```asp
res.setHeader(  'Set-Cookie',
                'username=xxx; 
                 path=/; 
                 httpOnly;
                 expires=${getCookieExpires()}'
             )
```
```asp
const getCookieExpires = () => {
  const d = new Date()
  d.setTime(d.getTime() + (10 * 24 * 60 * 60 * 1000))
  return d.toGMTString()
}
```
